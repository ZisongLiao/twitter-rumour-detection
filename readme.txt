Use "pip install simpletransformers" to install required module before compiling, the code requires NLTK module for tokenization and normolization, and BERT transformer for
text representation and classification.

The fine-tuning of rumour classifier requires Bert base uncased model, which can be found on https://github.com/google-research/bert.
